# Security Policy

## Supported Versions

As thinktank-ai doesn't provide an offical release yet, please use the latest version for the security updates.
Current we have two branches to maintain:
* main branch for thinktank-ai 2.x
* main-1.x branch for thinktank-ai 1.x

## Reporting a Vulnerability

Please go to https://github.com/thinktank-ai/thinktank-ai/security to report the vulnerability you find.
